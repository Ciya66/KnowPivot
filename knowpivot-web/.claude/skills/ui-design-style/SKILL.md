---

[项目类型] -企业级 AI 知识库平台 / Agentic 智能工作台 / 多轮对话 + 知识管理系统 / SaaS 风格 Web 应用

[整体风格]
现代极简（Modern Minimalism） + AI 科技感（AI Tech） + 企业级专业感（Enterprise-grade）
强调高信息密度但视觉清晰，偏工具型产品设计，弱装饰、强功能导向

[设计基调]
类 Notion + Linear + ChatGPT 的融合风格
结构清晰、留白充足、弱边框、卡片分层、强调内容可读性与交互效率

[配色方案]

- 主色：科技蓝 / 智能蓝（#3B82F6 / #2563EB）
- 辅助色：深灰（#1F2937）、中性灰（#6B7280）
- 背景：浅灰白（#F9FAFB）或深色模式（#0F172A）
- 强调色：紫色（AI感，#8B5CF6）用于智能功能 / Agent 行为提示
- 成功 / 警告 / 错误：标准语义色（绿 / 橙 / 红）
- 支持 浅色 + 深色双主题切换（必须）

[字体系统]

- 中文：思源黑体 / HarmonyOS Sans / 苹方
- 英文：Inter / SF Pro / Roboto
- 风格：无衬线、现代、清晰易读
- 强调：
  - 标题：SemiBold
  - 正文：Regular
  - 数据 / 代码：等宽字体（JetBrains Mono）

[组件细节]

- 布局结构：
  - 左侧导航（知识库 / Agent / 文档 / 设置）
  - 中间主工作区（对话 or 内容区）
  - 右侧辅助面板（引用文档 / 推理链 / 结果解释）
- 核心组件：
  - AI 对话窗口（类似 ChatGPT）
  - 文档卡片（支持预览 / 标签 / 版本）
  - 智能搜索框（语义检索）
  - Agent 执行状态条（步骤流 / thinking）
  - 标签系统（分类知识）
  - 数据卡片（总结 / 分析结果）
  - 文件上传区（拖拽上传）
  - 权限控制 UI（企业级）
- 视觉细节：
  - 圆角：8px / 12px
  - 阴影：轻阴影（层级分明但克制）
  - 边框：1px 极浅灰（弱化存在感）
  - 间距：8pt Grid 系统

[动效要求]

- 整体动效：流畅、克制、功能导向（类似 Linear 风格）
- 关键动效：
  - 对话打字机效果（AI 输出）
  - Agent “思考中” 动画（loading + steps）
  - 卡片 hover 微动效（轻微上浮 + 阴影变化）
  - 页面切换：淡入淡出（200-300ms）
  - 拖拽上传：高亮反馈
- 动效原则：
  - 不炫技，只增强“智能感”和“反馈感”

[产品关键词补充]Agentic RAG、私有化部署、企业知识库、多轮推理、智能问答、文档理解、自动总结、数据安全、低成本部署

[AI 交互与可解释性设计]设计需体现 AI 正在‘思考与调用知识’的过程，可视化 Agent 推理路径（如步骤流 / 引用来源 / reasoning 可视化）
